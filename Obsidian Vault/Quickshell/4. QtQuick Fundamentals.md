# QtQuick Fundamentals

> [!tip] Step 4 · Level 1
> QtQuick is the UI framework built on Qt6. You write QML, it renders with hardware acceleration. Everything you see — text, rectangles, images, layouts — comes from `import QtQuick`.

---

## Core Visual Types

### Item

Base of everything. Invisible, but has position, size, opacity, transforms.

```qml
Item {
  width: 100; height: 100
  opacity: 0.5
}
```

### Rectangle

Colored rect, optionally with rounded corners and a border.

```qml
Rectangle {
  width: 50; height: 50
  color: "#1e1e2e"
  radius: 8
  border { color: "white"; width: 2 }
}
```

### Text

Displays text. Supports rich text, fonts, wrapping.

```qml
Text {
  text: "Hello"
  color: "#cdd6f4"
  font { family: "JetBrainsMono Nerd Font"; pixelSize: 14 }
  horizontalAlignment: Text.AlignHCenter
}
```

### Image

Displays images from files or Qt resource system.

```qml
Image {
  source: "/path/to/image.png"
  fillMode: Image.PreserveAspectCrop
}
```

## Layout Types

### RowLayout / ColumnLayout

Position children in a row or column. Sizes are managed.

```qml
RowLayout {
  spacing: 8
  Text { text: "left" }
  Text { text: "center" }
  Text { text: "right" }
}
```

### GridLayout

Children arranged in a grid.

---

## Positioning & Anchoring

```qml
Item {
  Rectangle {
    anchors {
      top: parent.top
      left: parent.left
      right: parent.right
    }
    anchors.centerIn: parent       // center
    anchors.margins: 4             // margin on all sides
  }
}
```

Common anchors: `top`, `bottom`, `left`, `right`, `centerIn`, `fill`, `horizontalCenter`, `verticalCenter`

## Mouse & Touch

```qml
Rectangle {
  color: mouseArea.containsMouse ? "blue" : "red"
  MouseArea {
    id: mouseArea
    anchors.fill: parent
    hoverEnabled: true
    onClicked: console.log("clicked")
    onPressed: console.log("pressed")
    onReleased: console.log("released")
  }
}
```

## Animations

### PropertyAnimation (and friends)

```qml
Rectangle {
  NumberAnimation on opacity { from: 1; to: 0; duration: 200 }
}
```

### Transitions

```qml
Rectangle {
  Behavior on opacity {
    NumberAnimation { duration: 200; easing: Easing.OutCubic }
  }
}
```

### Easing Curves

- `Easing.Linear`, `Easing.OutCubic`, `Easing.OutBack` (bounce), `Easing.InCubic`
- Used in [[17. Bar Component]] tooltip animations

## Timer

```qml
Timer {
  interval: 1000     // ms
  running: true
  repeat: true
  onTriggered: root.refresh()
}
```

Used everywhere in your config for polling.

---

## Component.onCompleted

```qml
Rectangle {
  Component.onCompleted: console.log("ready!")
}
```

Runs once when the object is fully initialized.

## Key Patterns in Your Config

| QtQuick Type | Used Where |
|-------------|-----------|
| `Rectangle` | Bar background, tooltips, group boxes, workspace pills |
| `Text` | Clock, labels, icons (Nerd Font), everything |
| `Image` | System tray icons, app launcher icons |
| `RowLayout` | Bar center section (Clock + Workspaces + Controls) |
| `Item` | Group containers, clipping masks |
| `MouseArea` | Clickable icons, workspace pills, tray items |
| `Timer` | Polling battery, network, bluetooth, theme file, clock |
| `Behavior on X` | Tooltip show/hide, OSD animations |
| `NumberAnimation` | OSD fade, tooltip bounce |
