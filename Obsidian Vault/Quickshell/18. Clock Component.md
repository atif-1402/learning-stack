# Clock Component

> [!tip] Step 18 · Level 3
> A floating desktop clock overlay at `WlrLayer.Bottom`. Shows time, date, optional weather + pomodoro timer. Has a settings popup.

---

## Files

| File | Purpose |
|------|---------|
| `clock/shell.qml` | Full logic (538 lines) — clock window + settings popup + all features |
| `clock/config` | Saved settings (alignment, format, weather, pomodoro, font) |
| `clock/pos` | Saved position (bottom|right margins) |

---

## Clock Window — Key Structure

```qml
PanelWindow {
  WlrLayershell.layer: WlrLayer.Bottom
  // No anchors — user positions via drag
  // Reads pos file for saved position
  color: "transparent"
}
```

Features:
- **Day label** (e.g. "Tuesday")
- **Time** (HH:MM:SS or hh:mm AP, with seconds toggle)
- **Date** below time
- **Weather** (optional) — fetched from `wttr.in` every 30 min
- **Pomodoro timer** (optional) — 25 min focus / 5 min short break / 15 min long break

---

## Interaction

| Action | Effect |
|--------|--------|
| **Right-click + drag** | Reposition clock, saves to `pos` file |
| **Middle-click** | Toggle settings popup |
| **Settings popup** | Change alignment (left/center/right), 12/24h, seconds, font, weather, pomodoro |

---

## Settings Popup

A separate `PanelWindow` at `WlrLayer.Overlay` with:
- Alignment buttons (left/center/right)
- Hour format toggle (12h/24h)
- Seconds toggle
- Font family input
- Weather toggle + city name input
- Pomodoro toggle

Settings save to `clock/config` file.

---

## Pomodoro Timer

- 25 min focus → 5 min short break → repeat
- Every 4th break = 15 min long break
- Plays completion sound (via external command)
- Timer in settings: enable/disable

---

## Weather

- Fetches from `wttr.in/<city>`?format=j1 every 30 min
- Displays temp + condition icon
- City configurable in settings (default: Bhopal)
- Disabled by default

---

## Theme

Reads `foreground` and `accent` from `~/.config/omarchy/current/theme/colors.toml` using `grep` + `cut`. No poll timer — reads on settings open.

---

## Key Patterns

1. **Dragging**: Right-mouse drag updates window position, saves to file
2. **Persistent config**: Reads/writes `config` and `pos` files for state across restarts
3. **Dual windows**: Main clock at `WlrLayer.Bottom`, settings popup at `WlrLayer.Overlay`
4. **No Variants**: Single instance only (not per-screen)
