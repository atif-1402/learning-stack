# Quickshell API Modules Reference

> [!tip] Step 7 · Level 2
> Built-in integrations that ship with Quickshell. No external dependencies needed. This is a reference — skim it, then come back when you need a specific module.

| Type | Purpose | Used In |
|------|---------|---------|

## How Your Config Uses These

| Module | Active Component(s) |
|--------|-------------------|
| `Quickshell` | All (`shell.qml`, `Theme.qml`, etc.) |
| `Quickshell.Io` | All widgets that poll commands, Theme parsing, Frame config |
| `Quickshell.Hyprland` | Bar (workspaces, tray focus) |
| `Quickshell.Wayland` | All (layering, toplevel tracking for Frame) |
| `Quickshell.Services.SystemTray` | Bar (TrayGroup) |

Your config uses **external shell scripts** for audio, network, bluetooth, and battery instead of the native Pipewire/Networking/Bluetooth/UPower modules. The native modules could replace many of those polling shell commands.
