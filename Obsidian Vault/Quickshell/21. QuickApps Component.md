# QuickApps Component

> [!tip] Step 21 · Level 3
> Full-screen radial app launcher overlay. Apps arranged on a semicircular arc at the bottom of the screen.

---

## Files

| File | Purpose |
|------|---------|
| `quickapps/shell.qml` | Full logic (287 lines) — radial layout + keyboard/mouse nav |
| `quickapps/example.apps.json` | Example app list |

---

## Key Structure

```qml
PanelWindow {
  WlrLayershell.layer: WlrLayer.Overlay
  // Full screen
  // Dark background overlay at 88% opacity
  // Apps arranged in a semicircle at bottom
}
```

### Layout

Apps are positioned on a semicircular arc:
- Angles from ~160° to ~20° (bottom arc)
- Each app gets a tile with icon + name
- Radial position calculated with `Math.sin`/`Math.cos`

---

## Navigation

| Input | Action |
|-------|--------|
| Left/Right arrows or H/L | Move selection |
| Tab | Next app |
| Enter or Space | Launch selected app |
| 1-9 | Direct selection (by index) |
| Escape | Close launcher |
| Mouse hover | Select app |
| Mouse click | Launch app |

---

## Icons

- Loads from theme icon name (e.g. `alacritty`, `chromium`, `code`)
- Uses `MultiEffect` for desaturation/colorization effects
- Falls back to first letter of app name if icon fails to load

    { "name": "Browser",  "exec": "chromium",  "icon": "chromium",  "comment": "..." }

---

## Theme

Reads `background`, `foreground`, `accent`, `color8` from `~/.config/omarchy/current/theme/colors.toml` with TOML regex.

Background overlay is at 88% opacity for the dimming effect.

---

## Key Patterns

1. **Radial positioning** — math-based layout instead of standard QtQuick positioning
2. **MultiEffect** — GPU-accelerated image effects (desaturate, colorize)
3. **Keyboard-centric** — designed for fast keyboard navigation
4. **Overlay layer** — covers entire screen above all windows
