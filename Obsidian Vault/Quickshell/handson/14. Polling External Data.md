# 14 — Polling External Data

> [!tip] Step 14 · Hands-on
> Time: 10 minutes | Goal: Show battery percentage with icon

---

## Step 1: Create the file

```bash
nano ~/qs-tutorial/04-battery.qml
```

## Step 2: Write this

```qml
import Quickshell
import Quickshell.Io
import QtQuick

FloatingWindow {
  visible: true
  width: 200
  height: 80

  property int batteryPercent: 0
  property string batteryStatus: "Unknown"

  // Center content
  Row {
    anchors.centerIn: parent
    spacing: 8

    // Battery icon
    Text {
      id: batteryIcon
      anchors.verticalCenter: parent.verticalCenter
      font.family: "JetBrainsMono Nerd Font Propo"
      font.pixelSize: 24
      color: {
        if (batteryStatus === "Charging") return "#a6e3a1"
        if (batteryPercent <= 10) return "#f38ba8"
        if (batteryPercent <= 20) return "#f9e2af"
        return "#cdd6f4"
      }
      text: {
        if (batteryStatus === "Charging") return "\uf0e7"  // plug icon
        if (batteryPercent > 75) return "\uf240"
        if (batteryPercent > 50) return "\uf241"
        if (batteryPercent > 25) return "\uf242"
        return "\uf244"
      }
    }

    // Battery text
    Text {
      anchors.verticalCenter: parent.verticalCenter
      text: batteryPercent + "%"
      color: batteryIcon.color
      font.pixelSize: 18
    }
  }

  // Read battery capacity
  Process {
    id: capProc
    command: ["bash", "-c", "cat /sys/class/power_supply/BAT0/capacity 2>/dev/null || echo 0"]
    running: true
    stdout: StdioCollector {
      onStreamFinished: batteryPercent = parseInt(this.text.trim())
    }
  }

  // Read battery status
  Process {
    id: statProc
    command: ["bash", "-c", "cat /sys/class/power_supply/BAT0/status 2>/dev/null || echo Unknown"]
    running: true
    stdout: StdioCollector {
      onStreamFinished: batteryStatus = this.text.trim()
    }
  }

  // Timer to re-run both
  Timer {
    interval: 5000
    running: true
    repeat: true
    onTriggered: {
      capProc.running = true
      statProc.running = true
    }
  }
}
```

## Step 3: Run it

```bash
qs -p ~/qs-tutorial/04-battery.qml
```

You should see your battery percentage with a color-coded icon.

---

## What you learned

- `property int` — custom properties
- Multiple `Process` objects
- Conditional coloring with `color: { if ... }`
- Conditional icons for battery levels
- `parseInt()` — convert string to number

---

## Color Logic

| State | Color | Icon |
|-------|-------|------|
| Charging | Green (`#a6e3a1`) | Plug |
| ≤10% | Red (`#f38ba8`) | Empty |
| ≤20% | Yellow (`#f9e2af`) | Low |
| >75% | Default | Full |
| 25-75% | Default | Medium |
| ≤25% | Default | Low |
