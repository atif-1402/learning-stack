# 11 — Your First QML Window

> [!tip] Step 11 · Hands-on
> Time: 5 minutes | Goal: Get text on screen with Quickshell

---

## Step 1: Create a test file

```bash
mkdir -p ~/qs-tutorial
nano ~/qs-tutorial/01-hello.qml
```

## Step 2: Write this

```qml
import Quickshell
import QtQuick

FloatingWindow {
  visible: true
  width: 300
  height: 100

  Text {
    anchors.centerIn: parent
    text: "Hello, Quickshell!"
    color: "#cdd6f4"
    font.pixelSize: 20
  }
}
```

## Step 3: Run it

```bash
qs -p ~/qs-tutorial/01-hello.qml
```

You should see a floating window with "Hello, Quickshell!".

---

## What you learned

- `FloatingWindow` — a regular desktop window
- `Text` — displays text
- `anchors.centerIn: parent` — centers in parent
- Running with `qs -p`
