# 15 — Multi-Monitor Setup

> [!tip] Step 15 · Hands-on
> Time: 10 minutes | Goal: Create a bar that shows on every monitor

---

## Step 1: Create a config directory

```bash
mkdir -p ~/qs-tutorial/mybar
```

## Step 2: shell.qml

```bash
nano ~/qs-tutorial/mybar/shell.qml
```

```qml
import Quickshell
import QtQuick

ShellRoot {
  Variants {
    model: Quickshell.screens
    Bar {}
  }
}
```

## Step 3: Bar.qml

```bash
nano ~/qs-tutorial/mybar/Bar.qml
```

```qml
import Quickshell
import Quickshell.Io
import Quickshell.Wayland
import QtQuick
import QtQuick.Layouts

PanelWindow {
  required property var modelData
  screen: modelData
  visible: true

  anchors { top: true; left: true; right: true }
  implicitHeight: 32
  color: "#1e1e2e"
  WlrLayershell.layer: WlrLayer.Top
  exclusiveZone: implicitHeight

  RowLayout {
    anchors { left: parent.left; verticalCenter: parent.verticalCenter }
    spacing: 4

    // Workspace 1
    Rectangle {
      width: 24; height: 24; radius: 6
      color: "#cdd6f4"
      Text {
        anchors.centerIn: parent
        text: "1"
        color: "#1e1e2e"
        font.pixelSize: 12
      }
      MouseArea {
        anchors.fill: parent
        onClicked: console.log("switch to workspace 1")
      }
    }

    // Workspace 2
    Rectangle {
      width: 24; height: 24; radius: 6
      color: "#45475a"
      Text {
        anchors.centerIn: parent
        text: "2"
        color: "#cdd6f4"
        font.pixelSize: 12
      }
      MouseArea {
        anchors.fill: parent
        onClicked: console.log("switch to workspace 2")
      }
    }
  }

  // Center: clock
  Text {
    id: clockText
    anchors.centerIn: parent
    color: "#cdd6f4"
    font.pixelSize: 14

    Process {
      id: dateProc
      command: ["date", "+%H:%M"]
      running: true
      stdout: StdioCollector {
        onStreamFinished: clockText.text = this.text.trim()
      }
    }

    Timer {
      interval: 1000
      running: true
      repeat: true
      onTriggered: dateProc.running = true
    }
  }

  // Right: battery
  Text {
    anchors { right: parent.right; rightMargin: 8; verticalCenter: parent.verticalCenter }
    color: "#cdd6f4"
    font.pixelSize: 12

    Process {
      id: batProc
      command: ["bash", "-c", "cat /sys/class/power_supply/BAT0/capacity 2>/dev/null || echo 0"]
      running: true
      stdout: StdioCollector {
        onStreamFinished: parent.text = this.text.trim() + "%"
      }
    }

    Timer {
      interval: 30000
      running: true
      repeat: true
      onTriggered: batProc.running = true
    }
  }
}
```

## Step 4: Run it

```bash
qs -c ~/qs-tutorial/mybar
```

Each monitor gets its own bar with workspaces, clock, and battery.

---

## What you learned

- `ShellRoot` — config entry point
- `Variants { model: Quickshell.screens }` — one instance per screen
- `required property var modelData` — the screen data
- `screen: modelData` — assign window to correct monitor
- `exclusiveZone` — reserve space so windows don't overlap

---

## How multi-monitor works

```
Quickshell.screens = [Screen1, Screen2, Screen3]
        │
Variants creates:
  ├── PanelWindow { screen: Screen1 }   ← monitor 1
  ├── PanelWindow { screen: Screen2 }   ← monitor 2
  └── PanelWindow { screen: Screen3 }   ← monitor 3
```

Plug in a monitor → QS detects → new instance created.
Unplug → instance destroyed. All automatic.
