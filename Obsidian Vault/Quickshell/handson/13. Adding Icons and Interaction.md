# 13 — Adding Icons and Interaction

> [!tip] Step 13 · Hands-on
> Time: 10 minutes | Goal: Clickable Nerd Font icons with tooltips

---

## Step 1: Create the file

```bash
nano ~/qs-tutorial/03-icons.qml
```

## Step 2: Write this

```qml
import Quickshell
import QtQuick

FloatingWindow {
  visible: true
  width: 300
  height: 60

  Row {
    anchors.centerIn: parent
    spacing: 16

    // Volume icon
    Item {
      width: 40; height: 40

      Text {
        anchors.centerIn: parent
        text: "\uf028"  // Nerd Font volume icon
        font.family: "JetBrainsMono Nerd Font Propo"
        font.pixelSize: 20
        color: "#cdd6f4"
      }

      MouseArea {
        anchors.fill: parent
        hoverEnabled: true
        onClicked: console.log("volume clicked")
      }

      Rectangle {
        anchors.fill: parent
        color: "#cdd6f4"
        opacity: parent.MouseArea.containsMouse ? 0.1 : 0
        radius: 8
      }
    }

    // Bluetooth icon
    Item {
      width: 40; height: 40

      Text {
        anchors.centerIn: parent
        text: "\uf294"  // Nerd Font bluetooth icon
        font.family: "JetBrainsMono Nerd Font Propo"
        font.pixelSize: 20
        color: "#cdd6f4"
      }

      MouseArea {
        anchors.fill: parent
        hoverEnabled: true
        onClicked: console.log("bluetooth clicked")
      }

      Rectangle {
        anchors.fill: parent
        color: "#cdd6f4"
        opacity: parent.MouseArea.containsMouse ? 0.1 : 0
        radius: 8
      }
    }

    // WiFi icon
    Item {
      width: 40; height: 40

      Text {
        anchors.centerIn: parent
        text: "\uf1eb"  // Nerd Font wifi icon
        font.family: "JetBrainsMono Nerd Font Propo"
        font.pixelSize: 20
        color: "#cdd6f4"
      }

      MouseArea {
        anchors.fill: parent
        hoverEnabled: true
        onClicked: console.log("wifi clicked")
      }

      Rectangle {
        anchors.fill: parent
        color: "#cdd6f4"
        opacity: parent.MouseArea.containsMouse ? 0.1 : 0
        radius: 8
      }
    }
  }
}
```

## Step 3: Run it

```bash
qs -p ~/qs-tutorial/03-icons.qml
```

Hover over icons — they highlight. Click — console logs.

## Nerd Font Icon Reference

| Icon | Code | Name |
|------|------|------|
| Volume | `\uf028` | `nf-fa-volume_up` |
| Mute | `\uf026` | `nf-fa-volume_off` |
| Bluetooth | `\uf294` | `nf-fa-bluetooth` |
| WiFi | `\uf1eb` | `nf-fa-wifi` |
| Battery | `\uf240` | `nf-fa-battery_full` |
| Music | `\uf001` | `nf-fa-music` |
| Clock | `\uf017` | `nf-fa-clock_o` |
| Download | `\uf019` | `nf-fa-download` |

Search more at: [Nerd Font Cheatsheet](https://www.nerdfonts.com/cheat-sheet)

---

## What you learned

- Nerd Font icons via `font.family` + Unicode
- `MouseArea` with `hoverEnabled` for hover effects
- `Row` horizontal layout
- `opacity` binding for hover highlight
