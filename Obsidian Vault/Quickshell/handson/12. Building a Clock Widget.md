# 12 — Building a Clock Widget

> [!tip] Step 12 · Hands-on
> Time: 10 minutes | Goal: Show current time that updates every second

---

## Step 1: Create the file

```bash
nano ~/qs-tutorial/02-clock.qml
```

## Step 2: Write this

```qml
import Quickshell
import Quickshell.Io
import QtQuick

FloatingWindow {
  visible: true
  width: 200
  height: 80

  Text {
    id: clockText
    anchors.centerIn: parent
    color: "#cdd6f4"
    font.pixelSize: 24

    Process {
      id: dateProc
      command: ["date", "+%H:%M:%S"]
      running: true
      stdout: StdioCollector {
        onStreamFinished: clockText.text = this.text.trim()
      }
    }

    Timer {
      interval: 1000
      running: true
      repeat: true
      onTriggered: dateProc.running = true
    }
  }
}
```

## Step 3: Run it

```bash
qs -p ~/qs-tutorial/02-clock.qml
```

The time should update every second!

---

## What you learned

- `Process` — run external commands
- `StdioCollector` — read command output
- `Timer` — run code at intervals
- `dateProc.running = true` — re-run the process

---

## How it works

```
Timer (1s) → sets Process.running = true
  → Process runs "date +%H:%M:%S"
  → StdioCollector captures stdout
  → onStreamFinished: clockText.text = this.text
  → Text updates on screen
```

## Try it yourself

1. Change the date format (see `man date`)
2. Add the date below the time (hint: use another Process or a single `date` call)
3. Change the update interval to 5 seconds
